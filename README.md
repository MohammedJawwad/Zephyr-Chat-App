# ğŸ› ï¸ Installation  

## 1. Clone the repository:  
```bash
git clone https://github.com/MohammedJawwad/Zephyr-Chat-App.git
cd Zephyr-Chat-App
```
## 2. Install dependencies:
### For backend
```bash
cd backend
npm install
```
### For frontend
```bash
cd ../frontend
npm install
```
## 3. Start the application:
### Start backend
```bash
cd backend
npm start
```
### Start frontend
```bash
cd ../frontend
npm run dev
```
## 4. Access the app at http://localhost:3000.
